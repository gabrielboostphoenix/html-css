Principais Formatações de Textos

Negrito / Destaque

Nesta frase temos um termo em negrito usando as tags <b></b> (não semântica).

Nesta frase temos um termo em destaque usando as tags <strong></strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando as tags <i></i> (não semântica).

Nesta frase temos um termo em ênfase usando as tags <em></em> (semântica).

Texto Marcado

Podemos demarcar um texto para destacá-lo para fins de referência e notação usando as tags <mark></mark>.

As tags <mark></mark> desempenham a função de um marca texto.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto Excluído

Podemos indicar um texto que foi excluído, isto é, removido, ou melhor, deletado do conteúdo de um determinado documento usando as tags <del></del>.

Texto Inserido

Podemos também indicar que um texto foi adicionado, isto é, acrescentado ao conteúdo de um determinado documento usando as tags <ins></ins>.

Existe ainda o estilo de fonte sublinhado, como esse, usando as tags <u></u> (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo: x2 + 201/2 = 14

Texto Subscrito

Para inserir coisas do tipo: (fórmula da água) H2O
